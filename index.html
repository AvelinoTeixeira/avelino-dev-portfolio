O Processo Criativo

Pesquisa

Minhas decisões são guiadas por dados reais. Utilizo pesquisas para entender profundamente o usuário, criando soluções baseadas em insights confiáveis.

Estrategia

Planejo estratégias que combinem objetivos de negócio e UX, direcionando cada projeto para resultados impactantes alinhado com o cliente.

Design

Crio designs funcionais e atrativos que refletem a identidade da marca e otimizam o usuário experiência, com foco em inovação e usabilidade.

Portfólio

Dashboard

Este projeto de Dashboard visa criar um painel personalizável que centraliza dados essenciais, oferecendo uma visão em tempo real do desempenho empresarial para ajudar na tomada de decisões estratégicas.

VER PROJETO

Fincontrol

O "FinControl – Controle Financeiro Inteligente" é uma plataforma intuitiva de controle financeiro pessoal, que ajuda os usuários a gerenciar suas finanças, oferecendo informações claras sobre gastos, investimentos e metas financeiras.

VER PROJETO

WeatherNow

O WeatherNow é um aplicativo que fornece dados em tempo real sobre clima, com previsão de 5 dias, alertas para condições severas e geolocalização precisa, tudo com um design simples e intuitivo.

VER PROJETO

Headphones

Este projeto visa desenvolver e vender fones de ouvido de alta qualidade, oferecendo uma experiência sonora excepcional com design moderno e tecnologia avançada.

VER PROJETO

Whitepace

O projeto "WhitePace" oferece serviços de criação de sites, aplicativos e segurança, com soluções integradas para proteger ativos, informações e pessoas, visando proporcionar um ambiente seguro com tecnologias avançadas.

VER PROJETO

To-Do-List

Este projeto é uma aplicação de "Lista de Tarefas" em React JS, permitindo aos usuários gerenciar suas tarefas de forma eficiente com um design responsivo para dispositivos móveis e desktop.
